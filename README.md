# Gemini AI Telegram Bot
Простий та потужний Telegram-бот, інтегрований з моделлю штучного інтелекту Google Gemini. Бот дозволяє користувачам отримувати відповіді на будь-які запитання, а адміністратору — моніторити активність у реальному часі.

## Основні можливості

* Використовує модель `gemini-flash-lite-latest` для швидкої генерації відповідей;
* Автоматично розбиває довгі повідомлення (понад 4000 символів), щоб відповідати лімітам Telegram API;
* Бот автоматично надсилає адміністратору звіт про кожен запит (ім'я користувача, ID, текст питання та фрагмент відповіді);
* Підтримка зручного читання тексту від ШІ.

## Технології
* [Python 3.14.3](https://www.python.org/);
* pyTelegramBotAPI для роботи з Telegram;
* google-genai для доступу до Gemini;
* python-dotenv для керування змінними оточення.
